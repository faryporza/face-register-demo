# Framework System (ระบบและกรอบงานที่ใช้)

## Stack/Framework ที่เกี่ยวข้อง
- **Frontend:** Next.js (App Router), React, TypeScript, TailwindCSS  
- **Backend/API:** Next.js Route Handlers  
- **AI/Face:** face-api.js (Tiny Face Detector, Landmark Tiny, Face Recognition)  
- **Database:** MongoDB (Atlas) ผ่าน `MONGODB_URL`  
- **Auth:** Email/Password + Face Verification  
- **Deploy:** Vercel  

---

## Use Cases (กรณีการใช้งานหลัก)

### 1) ลงทะเบียนผู้ใช้ (Register)
- กรอกข้อมูลส่วนตัว (คำนำหน้า, ชื่อ, นามสกุล, เบอร์, อีเมล, รหัสผ่าน)
- ผ่านการตรวจสอบความถูกต้องของข้อมูล
- สแกนใบหน้า (หลายท่าทาง) + ตรวจระยะ + ตรวจความนิ่ง
- บันทึกข้อมูล + descriptor ลงฐานข้อมูล

### 2) เข้าสู่ระบบ (Login)
- กรอกอีเมล/รหัสผ่าน
- ยืนยันใบหน้าด้วยกล้อง (ต้องอยู่ในกรอบ + ใกล้พอ + นิ่งต่อเนื่อง)
- เข้าหน้า Home เมื่อยืนยันสำเร็จ

### 3) เข้าสู่ระบบด้วยหน้ากาก (Login with Mask)
- กรอกอีเมล/รหัสผ่าน
- ยืนยันใบหน้าขณะสวมหน้ากาก (threshold ปรับสูงขึ้นเล็กน้อย)
- ต้องอยู่ในกรอบ + ใกล้พอ + นิ่งต่อเนื่องก่อนผ่าน
- เข้าหน้า Home เมื่อยืนยันสำเร็จ

### 4) เช็คอิน (Check-in)
- ตรวจใบหน้าและบันทึกเวลาเข้า
- มี cooldown กันบันทึกซ้ำถี่เกินไป

### 5) ดูภาพรวม (Dashboard)
- แสดงสรุปจำนวน log ทั้งหมด
- กราฟสถิติรายวัน
- รายการล่าสุด

### 6) ข้อมูลส่วนตัว (Profile)
- แสดงข้อมูลผู้ใช้ที่ล็อกอิน

### 7) ประวัติการลงเวลา (Logs)
- ดูประวัติ Check-in/Check-out แบบแบ่งหน้า (Pagination)

### 8) จัดการผู้ใช้ (Admin)
- แสดงรายชื่อผู้ใช้แบบแบ่งหน้า
- แก้ role / ลบผู้ใช้
- เข้าถึงได้เฉพาะ admin
